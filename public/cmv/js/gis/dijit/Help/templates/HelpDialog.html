Map navigation using mouse and keyboard: Navigasi peta yang digunakan oleh mouse dan keyboard:
  • Menggeser Objek
  • SHIFT + Click untuk mentengahkan peta
  • SHIFT + seret untuk memperbesar peta
  • SHIFT + CTRL + seret untuk memperkecil peta
  • geser mouse scroll ke depan untuk memperbesar
  • geser mouse scroll ke belakang untuk memperkecil
  • + untuk memperbesar level peta
  • - untuk memperkecil level peta
  • Klik 2 kali untuk metengahkan dan memperbesar peta.
Berbagai pencarian dapat dilakukan di kotak pencarian:
  • Cari bedasarkan Alamat
  • Cari bedasarkan nama tempat
Selain kemampuan pencarian, alat-alat berikut disediakan:
  • Cari
    • Alat untuk mencari fasilitas - fasilitas yang tersedia.
  • Alat Ukur
    • Alat ukur menyediakan kemampuan untuk menggambar titik, garis, atau poligon pada peta dan menentukan unit pengukuran.
  • Cetak Peta
    • Peta ini dapat diekspor ke berbagai format dan layout.
  • Lapisan Peta
    • Menampilkan layer - layer peta yang tersedia.
  • Legenda
    • Menampilkan informasi dari peta.
  • Petunjuk Arah
    • Alat ini untuk mencari rute terdekat dari lokasi A ke lokasi B.
  • Editor
    • Alat ini untuk membuat geotagging pada peta.
  • Cara Penggunaan
