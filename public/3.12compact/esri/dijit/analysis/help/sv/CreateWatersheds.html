Skapa avrinningsområden

Det här verktyget fastställer avrinningsområdet, eller flödesackumuleringsområdet uppströms, för varje punkt i analyslagret.

Du kan till exempel ha punktgeoobjekt som representerar platser för vattenburna föroreningar och vill hitta de troliga källorna till föroreningen. Eftersom källan till föroreningen måste ligga någonstans i avrinningsområdet uppströms från punkten, kan du använda det här verktyget för att avgöra vilket avrinningsområde som innehåller källorna till föroreningen.

Platsen för analyspunkterna är viktig. Om de inte ligger exakt på en dräneringslinje kan de resulterande avrinningsområdena bli mycket små och inte särskilt användbara för din analys. Sökavstånd till närmaste dränering kan användas för att justera platsen för dina analyspunkter så att de hamnar precis på en dräneringslinje. Därmed kan du hitta det avrinningsområde som rinner till punkten på dräneringslinjen. De justerade platserna som används för att hitta avrinningsområdena skrivs till utdatalagret – dina indataanalyspunkter ändras inte.

Resultatet blir ett grupplager som innehåller två lager: avrinningsområdena och de justerade punkter som används för att beräkna avrinningsområdena.

De resulterande avrinningsområdena kommer att ha alla attribut från indatapunkterna tillsammans med följande ytterligare attribut:

  • PourPtID– innehåller ID-värdet från indatapunkten
  • Area Square Kilometers– innehåller området i kvadratkilometer från avrinningsområdet uppströms
  • Description samt DataResolution – innehåller information som rör de hydrologiska data som används för att beräkna spårningen

Om Använd aktuell kartutbredning är markerat används endast de geoobjekt som finns i den aktuella kartutbredningen vid beräkningen av avrinningsområdena. Om alternativet inte markeras analyseras alla geoobjekt i analyslagret, även om de ligger utanför den aktuella kartutbredningen.

Sökavstånd till närmaste dränering

Om analyspunkterna ligger långt från en dräneringslinje, blir de resulterande avrinningsområdena troligtvis mycket små och är inte till mycket nytta för att avgöra källan till föroreningen uppströms. I de flesta fall vill du att analyspunkterna ska snappa till närmaste dräneringslinje för att hitta de avrinningsområden som rinner till en punkt som finns längs dräneringslinjen. Ange ett sökavstånd om du vill hitta den närmaste dräneringslinjen. Om du inte anger ett sökavstånd kommer verktyget att beräkna och använda ett konservativt sökavstånd.

Om du vill använda den exakta platsen för din indatapunkt anger du sökavståndet noll.

Dräneringslinjerna har i analyssyfte beräknats i förväg av Esri med hjälp av hydrologiska standardmodeller. Om det inte finns någon dräneringslinje inom sökavståndet används den plats som innehåller den största flödesackumuleringen inom sökavståndet.

Resultatlagrets namn

Det här är namnet på grupplagret som skapas i Mitt innehåll och läggs till i kartan. Grupplagret innehåller ett lager för avrinningsområdena och de justerade punkterna. Om lagret redan finns uppmanas du bekräfta att du vill skriva över det.

Om du använder listrutan Spara resultat i kan du ange namnet på en mapp i Mitt innehåll där resultatet ska sparas.
