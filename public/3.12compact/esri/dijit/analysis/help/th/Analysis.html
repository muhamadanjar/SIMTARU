สรุปรวมข้อมูล

เครื่องมือเหล่านี้ใช้สำหรับคำนวณค่าทั้งหมด ความยาว และรายละเอียดพื้นฐานจากค่าสถิติของฟีเจอร์และข้อมูลคุณลักษณะที่อยู่ภายในบริเวณหรือใกล้กับฟีเจอร์อื่นๆ

  • รวมจุด คำนวณค่าสถิติจากจุดที่อยู่ภายในบริเวณที่กำหนด
  • การสรุปโดยระยะทาง คำนวณค่าสถิติของฟีเจอร์หรือข้อมูลคุณลักษณะที่อยู่ภายในระยะทางที่กำหนด
  • การสรุปข้อมูลที่อยู่ภายใน คำนวณค่าสถิติของพื้นที่ที่สนใจหรือข้อมูลคุณลักษณะที่มีการซ้อนทับกัน

รวมจุด

การใช้ชั้นข้อมูลประเภทจุดและชั้นข้อมูลประเภทพื้นที่ เครื่องมือนี้จะทำการเลือกจุดที่อยู่ในแต่ละพื้นที่และคำนวณค่าทางสถิติของจุดทุกจุดที่อยู่ในพื้นที่นั้นๆ ตัวอย่างเช่น:

  • แสดงจุดเกิดเหตุอาชญากรรม แล้วนับจำนวนอาชญากรรมที่เกิดขึ้นในหนึ่งเขตการปกครองหรือในท้องที่นั้นๆ
  • ค้นหากำไรสูงสุดและต่ำสุดที่ได้จากยอดขายสาขาแฟรนไชส์ในแต่ละรัฐ


สรุปใกล้เคียง

ค้นหาฟีเจอร์ที่อยู่ในระยะที่กำหนดของฟีเจอร์ที่มีการวิเคราะห์ชั้นข้อมูล ระยะทางนั้นสามารถวัดตามความยาวของทางที่เป็นเส้นตรงหรือเส้นทางผันแปรของรูปแบบการเดินทาง และค่าสถิติที่ถูกคำนวณโดยใช้ฟีเจอร์ที่อยู่ใกล้เคียง ตัวอย่างเช่น:

  • คำนวณหาจำนวนประชากรทั้งหมดที่อยู่ในระยะเวลาการเดินทาง 5 นาทีจากตำแหน่งร้านค้าที่จะเปิดใหม่
  • คำนวณจำนวนทางหลวงที่สามารถเข้าถึงตำแหน่งร้านค้าเปิดใหม่ได้ภายใน 5 นาที โดยการวัดความสามารถในการเข้าถึงร้านค้า


สรุปภายใน

ค้นหาพื้นที่ (และบางส่วนของพื้นที่) ที่ซ้อนทับกันระหว่างชั้นข้อมูล 2 ชั้น และคำนวณค่าสถิติที่มีการซ้อนทับกัน ตัวอย่างเช่น:

  • กำหนดชั้นข้อมูลพื้นที่รับน้ำและชั้นข้อมูลการใช้ที่ดินตามประเภทที่ดิน แล้วคำนวณพื้นที่การใช้ที่ดินทั้งหมดในแต่ละพื้นที่รับน้ำ
  • กำหนดชั้นข้อมูลพื้นที่เมืองและชั้นข้อมูลขอบเขตเมือง แล้วสรุปค่าเฉลี่ยของพื้นที่ว่างเปล่าที่มีในแต่ละเมือง


การปรับแต่งข้อมูลเพิ่มเติม

เครื่องมือเหล่านี้ช่วยคุณในการสำรวจลักษณะของพื้นที่ รายละเอียดเกี่ยวกับข้อมูลประชากรและสถิติถูกส่งกลับไปยังพื้นที่ที่คุณได้เลือกไว้ นอกจากนี้ ข้อมูลเปรียบเทียบยังสามารถส่งรายงานไปยังพื้นที่ที่ต่อขยาย เช่น ประเทศและรัฐ ได้

  • การปรับแต่งชั้นข้อมูลเพิ่มเติม เรียกคืนข้อมูลเกี่ยวกับคน สถานที่ และธุรกิจในพื้นที่เฉพาะ หรือในระยะทางตรง หรือจากระยะเวลาระหว่างที่หนึ่งไปยังอีกที่หนึ่ง

การปรับแต่งข้อมูลเพิ่มเติม

เรียกคืนข้อมูลเกี่ยวกับคน สถานที่ และธุรกิจในพื้นที่เฉพาะ หรือในระยะทางตรง หรือจากระยะเวลาระหว่างที่หนึ่งไปยังอีกที่หนึ่ง


ค้นหาตำแหน่ง

เครื่องมือนี้ใช้ในการระบุพื้นที่ที่ตรงกับจำนวนเงื่อนไขที่คุณระบุไว้ต่างๆ กัน เงื่อนไขเหล่านี้ขึ้นอยู่กับคำสั่งเชิงคุณลักษณะ (เช่น แปลงที่ดินที่ว่างเปล่า) และคำสั่งเชิงพื้นที่ (เช่น ภายในระยะทาง 1 กิโลเมตรจากแม่น้ำ) พื้นที่ที่ค้นพบสามารถเลือกได้จากฟีเจอร์ที่มีอยู่แล้ว (เช่น แปลงที่ดินที่มีอยู่แล้ว) หรือฟีเจอร์ใหม่ที่สร้างขึ้นเพื่อให้สามารถตอบโจทย์ได้ทุกความต้องการ

  • ค้นหาตำแหน่งที่มีอยู่แล้ว ค้นหาพื้นที่ที่มีอยู่จากการวิเคราะห์ชั้นข้อมูลเพื่อให้สอดคล้องกับเงื่อนไขต่างๆ ที่กำหนดไว้
  • ค้นหาตำแหน่งใหม่จากข้อมูลที่มีอยู่ สร้างพื้นที่ขึ้นมาใหม่จากตำแหน่งที่คุณกำลังศึกษาและสอดคล้องกับเงื่อนไขต่างๆ ที่กำหนดไว้
  • ค้นหาตำแหน่งที่ใกล้เคียง ค้นหาตำแหน่งที่ใกล้เคียงกับอีกตำแหน่งหนึ่งหรือตำแหน่งอื่นๆ ที่อ้างอิงจากเงื่อนไขที่คุณกำหนดขึ้น
  • สร้างส่วนที่สายตามองเห็น สร้างพื้นที่ส่วนที่มองเห็นได้โดยอิงจากตำแหน่งที่คุณระบุ
  • สร้างพื้นที่รับน้ำ สร้างพื้นที่รับน้ำโดยอิงจากตำแหน่งที่คุณระบุ
  • ค้นหาตามน้ำ กำหนดเส้นทางการไหลในทิศทางไหลลงจากตำแหน่งที่คุณระบุ

ค้นหาตำแหน่งที่มีอยู่

เลือกฟีเจอร์ที่มีอยู่จากพื้นที่ที่คุณกำลังศึกษาเพื่อให้สอดคล้องกับชุดเงื่อนไขที่คุณกำหนดไว้ เงื่อนไขเหล่านี้ขึ้นอยู่กับคำสั่งเชิงคุณลักษณะ (เช่น แปลงที่ดินที่ว่างเปล่า) และคำสั่งเชิงพื้นที่ (เช่น ภายในระยะทาง 1 กิโลเมตรจากแม่น้ำ)


ค้นหาตำแหน่งใหม่จากข้อมูลที่มีอยู่

เลือกฟีเจอร์ที่มีอยู่จากพื้นที่ที่คุณกำลังศึกษาเพื่อให้สอดคล้องกับชุดเงื่อนไขที่คุณกำหนดไว้ เงื่อนไขเหล่านี้ขึ้นอยู่กับคำสั่งเชิงคุณลักษณะ (เช่น แปลงที่ดินที่ว่างเปล่า) และคำสั่งเชิงพื้นที่ (เช่น ภายในระยะทาง 1 กิโลเมตรจากแม่น้ำ)


ค้นหาตำแหน่งที่ใกล้เคียง

ค้นหาตำแหน่งที่ใกล้เคียงกับอีกตำแหน่งหนึ่งหรือตำแหน่งอื่นๆ ที่อ้างอิงจากเงื่อนไขที่คุณกำหนดขึ้น


สร้างส่วนที่สายตามองเห็น

สร้างพื้นที่ส่วนที่มองเห็นได้โดยอิงจากตำแหน่งที่คุณระบุ


สร้างพื้นที่รับน้ำ

สร้างพื้นที่รับน้ำโดยอิงจากตำแหน่งที่คุณระบุ


ค้นหาตามน้ำ

กำหนดเส้นทางการไหลในทิศทางไหลลงจากตำแหน่งที่คุณระบุ


รูปแบบการวิเคราะห์

เครื่องมือเหล่านี้มีหน้าที่ในการระบุตัวตน คำนวณปริมาณ และแสดงผลลัพธ์ข้อมูลของคุณในรูปแบบเชิงพื้นที่

  • คำนวณความหนาแน่น ตอบคำถามเกี่ยวกับจำนวนเหตุการณ์และการกระจายตัวของเหตุการณ์เหล่านั้นผ่านแผนที่
  • ค้นหาจุดฮอตสปอต ระบุจากข้อมูลรูปแบบเชิงพื้นที่ที่ได้จากการรวบกลุ่มทางสถิติที่สำคัญ
  • เพิ่มจุดข้อมูล คาดการณ์ข้อมูลที่จะเกิดขึ้นในตำแหน่งใหม่ๆ โดยอ้างอิงจากระยะข้อมูลที่พบในชุดข้อมูลที่สะสมไว้

คำนวณความหนาแน่น

การวิเคราะห์หาความหนาแน่น เป็นการตอบคำถามเกี่ยวกับจำนวนเหตุการณ์และการกระจายตัวของเหตุการณ์เหล่านั้นผ่านแผนที่ ตัวอย่างในการใช้เครื่องมือนี้ ได้แก่ การแสดงจำนวนการเกิดฟ้าผ่าหรือพายุทอร์นาโด การเข้าถึงการสิ่งอำนวยความสะดวกในการดูแลสุขภาพ และความหนาแน่นประชากร


ค้นหาจุดสูงสุด

เครื่องมือนี้ใช้ในการสร้างแผนที่สำหรับแสดงข้อมูลเชิงพื้นที่ที่ได้จากการรวบข้อมูลทางสถิติที่สำคัญ ใช้เครื่องมือนี้ในการยกเลิกจุดที่ไม่ต้องการ ทั้งจุดที่มีความถี่สูงสุด(สีแดง) และจุดที่มีความถี่น้อยสุด (สีน้ำเงิน) ไม่ว่าจะเป็นค่าใดก็ตาม เช่น มูลค่าบ้านเรือน ความถี่ในการเกิดอุบัติเหตุบนท้องถนน อัตราการจ้างงาน หรือค่าความหลากหลายทางชีวภาพ เป็นต้น


เพิ่มจุดข้อมูล

ด้วยเครื่องมือนี้ คุณสามารถคาดการณ์ข้อมูลที่จะเกิดขึ้นในตำแหน่งใหม่ๆ ได้ โดยอ้างอิงจากระยะข้อมูลที่พบในชุดข้อมูลที่สะสมไว้ โดยเครื่องมือนี้จะนำเอาข้อมูลจุดที่มีค่าตัวแปรแต่ละค่า และแก้ไขพื้นที่ให้เป็นไปตามกลุ่มที่มีการคาดการณ์ค่าตัวแปรไว้ ตัวอย่างในการใช้เครื่องมือนี้ ได้แก่ ใช้คาดการณ์ระดับปริมาณฝนในพื้นที่รับน้ำ โดยอ้างอิงจากปริมาณระดับน้ำฝนที่วัดได้ในแต่ละจุด


ใช้ระยะใกล้เคียง

เครื่องมือเหล่านี้ช่วยให้คุณตอบคำถามทั่วไปที่เกี่ยวกับการวิเคราะห์เชิงพื้นที่ได้: "อะไรใกล้กับอะไร"

  • การเชื่อมโยงจากจุดเริ่มต้นไปยังจุดหมายปลายทาง ช่วยวัดระยะเวลาในการเดินทางหรือระยะทางระหว่างจุดสองจุด
  • การสร้างระยะกันชน ใช้ในการสร้างพื้นที่ที่ออกจากจุดสนใจเป็นระยะทางเท่าๆ กัน
  • การสร้างพื้นที่จากระยะเวลาการขับขี่ ค้นหาพื้นที่รอบๆ ตำแหน่งที่ต้องการเข้าถึงภายในระยะเวลาที่กำหนด
  • การค้นหาระยะที่ใกล้ที่สุด กำหนดตำแหน่งต่างๆ ที่ใกล้กับตำแหน่งที่เลือกมากที่สุด
  • การวางแผนเส้นทาง พิจารณาเลือกเส้นทางที่ดีที่สุดในการเดินทางไปยังจุดต่างๆ โดยใช้พาหนะต่างๆ

เชื่อมต่อจุดเริ่มต้นไปยังจุดสิ้นสุด

การเชื่อมโยงจากจุดเริ่มต้นไปยังจุดหมายปลายทางช่วยวัดระยะเวลาในการเดินทางหรือระยะทางระหว่างจุดสองจุด เครื่องมือนี้สามารถรายงานระยะทางเส้นตรง เส้นทางถนน หรือระยะเวลาในการเดินทาง คุณสามารถใช้งานเครื่องมือนี้ได้ดังนี้

  • คำนวณระยะทางหรือเวลาโดยรวมของคนเดินทางที่ใช้เดินทางไปกลับระหว่างบ้านและที่ทำงาน
  • คำนวณระยะทางที่ลูกค้าจะเดินทางเพื่อไปจับจ่ายซื้อของที่ร้านค้าของคุณ ใช้ข้อมูลนี้ในการระบุกลุ่มตลาดที่ต้องการ โดยเฉพาะการสร้างแคมเปญโฆษณาหรือการเลือกตำแหน่งที่ตั้งร้านค้าใหม่
  • คำนวณระยะทางในการเดินทางที่คาดไว้สำหรับยานพาหนะของคุณ หลังจากนั้น ทำการประมวลผลด้วยเครื่องมือ การสรุปผลภายใน เพื่อรายงานระยะทางภายในเมืองหรือภูมิภาคอื่นๆ

คุณจัดเตรียมข้อมูลจุดเริ่มต้นและจุดสิ้นสุด และเครื่องมือนี้จะปรับชั้นข้อมูลที่ประกอบด้วยเส้นถนน และเครื่องมือวัด ระหว่างจุดเริ่มต้นและจุดหมายปลายทาง


การสร้างระยะกันชน

ระยะกันชน คือ พื้นที่ที่ครอบคลุมทั้งข้อมูลจุด เส้น หรือพื้นที่ที่สนใจ โดยมีระยะทางที่ขยายออกไปเท่าๆ กัน

ระยะกันชน มักใช้สร้างพื้นที่ใหม่ที่สามารถนำไปวิเคราะห์ร่วมกับเครื่องมืออื่นๆ ได้ เช่น ชั้นข้อมูลซ้อนทับ ตัวอย่างเช่น จากคำถาม "ในระยะ 1 ไมล์จากโรงเรียนนี้ มีสิ่งก่อสร้างใดบ้าง" คุณสามารถหาคำตอบได้จากการสร้างระยะกันชน 1 ไมล์ จากโรงเรียน และทำการซ้อนทับระหว่างระยะกันชนกับชั้นข้อมูลที่มีสิ่งก่อสร้างต่างๆ ผลลัพธ์ที่ได้คือ ชั้นข้อมูลแสดงสิ่งก่อสร้างที่อยู่ในระยะ 1 ไมล์จากโรงเรียน


สร้างขอบเขตจากระยะเวลาขับขี่

พื้นที่ที่ได้จากระยะเวลาการขับรถ คือ พื้นที่ที่สามารถเข้าถึงได้ในระยะเวลาหรือระยะทางที่ได้จากการขับขี่ที่กำหนด พื้นที่ดังกล่าวนี้ ช่วยให้คุณตอบคำถามเหล่านี้ได้:

  • สถานที่ใดที่ฉันสามารถไปจากตรงนี้ภายในการขับรถ 30 นาที
  • ฉันสามารถขับรถไปที่ใดได้บ้างในระยะเวลา 30 นาที ในช่วงเวลาเร่งด่วนอย่างนี้
  • พื้นที่ใดในเมืองที่เจ้าหน้าที่ดับเพลิงไปถึงภายใน 5 นาที การปรับปรุงการให้ความช่วยเหลือในเหตุการณ์ไฟไหม้ โดยการสร้างสถานีดับเพลิงใหม่ที่นี่จะทำได้อย่างไร
  • ธุรกิจของฉันครอบคลุมพื้นที่การตลาดใดบ้าง มีพื้นที่ใดบ้างในระยะ 10 ไมล์จากร้านค้าของฉัน


ค้นหาที่ใกล้เคียงที่สุด

คำนวณค่าใช้จ่ายในการท่องเที่ยวว่าแบบใดใกล้เคียงกันมากที่สุด ระหว่างแบบไม่เตรียมการ แบบมีสิ่งอำนวยความสะดวก และแบบเตรียมพร้อมทุกอย่าง ผลลัพธ์ที่ได้คือ ชั้นข้อมูลที่แสดงเส้นทางที่ดีที่สุดระหว่างแบบไม่เตรียมการและแบบมีสิ่งอำนวยความสะดวก พร้อมกับค่าใช้จ่ายระหว่างการเดินทาง (เวลาและระยะทาง) ในแต่ละเส้นทาง ตัวอย่างเช่น คุณสามารถใช้เครื่องมือนี้ค้นหาโรงพยาบาลที่ใกล้ที่สุดในกรณีที่เกิดอุบัติเหตุ หรือตู้ ATM ที่ใกล้ที่สุดจากจุดที่คุณอยู่ ณ ตอนนี้


วางแผนเส้นทาง

คุณจัดเตรียมชุดข้อมูลประเภทตำแหน่งที่ต้องหยุด และจำนวนยานพานหะที่ใช้งานจริง โดยการวางแผนเส้นทางจะช่วยพิจารณาจุดที่พาหนะควรหยุดหรือเส้นทางที่พาหนะหยุดแล้วจะเกิดประสิทธิภาพมากที่สุด

ใช้เครื่องมือนี้ในการวางแผนการทำงานสำหรับทีมสำรวจ ทีมประเมินราคา ทีมผู้ให้บริการถึงบ้านลูกค้า และอื่นๆ ทั้งการบริการส่งหรือการไปรับสินค้าจากตำแหน่งที่ห่างไกล หรือการบริการขนส่งมวลชน


จัดการข้อมูล

เครื่องมือเหล่านี้ใช้ได้กับทั้งการจัดการข้อมูลทางภูมิศาสตร์แบบวันต่อวันและแบบรวบรวมข้อมูลก่อนที่จะนำไปวิเคราะห์

  • การรวมขอบเขตของข้อมูล เชื่อมพื้นที่ที่มีขอบเขตและข้อมูลในตารางคุณลักษณะทั่วไปร่วมกัน
  • การค้ดลอกข้อมูล สร้างชุดข้อมูลใหม่โดยการคัดลอกฟีเจอร์จากข้อมูลที่คุณมีอยู่แล้ว
  • Merge Layers copies all the features from two or more existing layers into a new layer.
  • Overlay Layers combines two or more layers into one single layer. You can think of overlay as peering through a stack of maps and creating a single map containing all the information found in the stack.

การรวมขอบเขตพื้นที่

พื้นที่ที่ซ้อนทับกันหรือมีการใช้ขอบเขตร่วมกัน จะถูกผสานเข้าด้วยกันให้เป็นพื้นที่เดียวกัน

คุณสามารถกำหนดขอบเขตที่ต้องการทำการรวมข้อมูลได้ด้วยการกำหนดที่ฟิลด์ For example, if you have a layer of counties, and each county has a State_Name attribute, you can dissolve boundaries using the State_Name attribute. Adjacent counties will be merged together if they have the same value for State_Name. ผลลัพธ์ที่ได้คือชั้นข้อมูลที่แสดงขอบเขตในระดับเขต


สร้างข้อมูล

Creates a zip file, CSV, or KMZ of data from your layers and an area of interest that you specify.


การรวมชั้นข้อมูล

Copies features from two or more existing layers into a new layer, for example:

  • I have three layers for England, Wales, and Scotland and I want a single layer of Great Britain.
  • I have 12 layers and each contains parcel information for contiguous townships. I want to join them together into a single layer, keeping only the attributes that have the same name and type on the 12 input layers.


การซ้อนทับชั้นข้อมูล

การซ้อนทับชั้นข้อมูล คือการผสมผสานชั้นข้อมูล 2 ชั้น หรือมากกว่านั้นให้เหลือเพียงชั้นข้อมูลเดียว You can think of overlay as peering through a stack of maps and creating a single map containing all the information found in the stack, for example:

  • What parcels are within the 100-year floodplain? (Within is just another way of saying on top of.)
  • มีถนนสายใดบ้างที่อยู่ในเมืองนี้?
  • ดินประเภทนี้ มีการใช้ที่ดินอย่างไรบ้าง?
  • มีอะไรหลงเหลืออยู่ในหลุมของทหารที่ถูกทิ้งร้างบ้าง?
