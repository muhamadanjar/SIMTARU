สร้างพื้นที่รับน้ำ

This tool determines the watershed, or upstream contributing area, for each point in your analysis layer.

ตัวอย่างเช่น สมมุติว่าคุณมีจุดที่แทนตำแหน่งของการปนเปื้อนที่มากับน้ำ และคุณต้องการค้นหาแหล่งของการปนเปื้อนนั้น เนื่องจากแหล่งของการปนเปื้อนต้องเป็นที่ใดที่หนึ่งภายในเส้นทางการไหลจากต้นน้ำของจุดนั้น คุณควรใช้เครื่องมือนี้เพื่อระบุพื้นที่รับน้ำที่มีแหล่งของการปนเปื้อน

ตำแหน่งของจุดวิเคราะห์นั้นมีความสำคัญ หากจุดที่คุณวิเคราะห์ไม่ได้อยู่บนเส้่นทางระบายน้ำ พื้นที่รับน้ำที่ได้นั้นอาจเล็กมากและไม่มีประโยชน์ในงานวิเคราะห์ของคุณ สามารถนำ การค้นหาระยะทางไปยังเส้นทางระบายน้ำที่ใกล้ที่สุด มาใช้ได้เพื่อปรับตำแหน่งของจุดวิเคราะห์ของคุณ เพื่อให้อยู่ภายในช่วงของเส้นทางระบายน้ำจริงๆ ในการค้นหาพื้นที่รับน้ำที่ไหลลงสู่จุดที่อยู่บนเส้นทางระบายน้ำ ตำแหน่งที่ปรับซึ่งนำมาใช้ค้นหาพื้นที่รับน้ำจะถูกเขียนลงบนชั้นข้อมูลที่ได้—จุดวิเคราะห์ข้อมูลที่คุณป้อนเข้ามาจะไม่เปลี่ยนแปลง

ผลลัพธ์ที่ได้คือ กลุ่มชั้นข้อมูลที่ประกอบด้วยชั้นข้อมูลสองชั้น: พื้นที่รับน้ำและจุดที่ถูกปรับซึ่งนำมาใช้คำนวณพื้นที่รับน้ำ

พื้นที่รับน้ำที่ได้จะมีรายละเอียดทั้งหมดของจุดที่คุณป้อนข้อมูล รวมถึงรายละเอียดเพิ่มเติมเหล่านี้:

  • PourPtID—ประกอบด้วยค่า ID จากจุดป้อนข้อมูล
  • Area Square Kilometers—ประกอบด้วยพื้นที่ที่มีหน่วยเป็นตารางกิโลเมตรของพื้นที่รับน้ำจากต้นน้ำ
  • Description และ DataResolution—ประกอบด้วยข้อมูลจำเพาะทางอุทกวิทยาที่นำมาใช้คำนวณทางน้ำ

ถ้าเลือก ใช้ขอบเขตแผนที่ปัจจุบัน เฉพาะคุณลักษณะที่แสดงในขอบเขตแผนที่ปัจจุบันเท่านั้นที่จะนำมาใช้คำนวณพื้นที่รับน้ำ ถ้าไม่เลือก คุณลักษณะทั้งหมดในชั้นข้อมูลวิเคราะห์จะถูกนำไปวิเคราะห์ แม้ว่าจะอยู่นอกขอบเขตแผนที่ปัจจุบัน

ค้นหาระยะทางไปยังเส้นทางระบายน้ำที่ใกล้ที่สุด

หากจุดวิเคราะห์ของคุณอยู่ห่างจากทางระบายน้ำ พื้นที่รับน้ำที่ได้น่าจะมีขนาดเล็กมากและไำม่ค่อยมีประโยชน์ในการตรวจสอบแหล่งของการปนเปื้อนจากต้นน้ำ ในเกือบทุกกรณี คุณต้องการจุดวิเคราะห์ในการเจาะเข้าสู่ทางระบายน้ำที่อยู่ใกล้ที่สุด เพื่อค้นหาพื้นที่รับน้ำที่ไหลไปยังจุดที่อยู่ในทางระบายน้ำ เมื่อต้องการค้นหาทางระบายน้ำที่อยู่ใกล้ที่สุด ให้ระบุระยะทางที่จะใช้ค้นหา หากคุณไม่ได้ระบุระยะทางที่จะใช้ค้นหา เครื่องมือจะคำนวณและใช้ระยะทางค้นหาที่เก็บเอาไว้

เมื่อต้องการใช้ตำแหน่งที่แน่นอนของจุดที่คุณป้อนข้อมูล ให้กำหนดระยะทางที่จะใช้ค้นหาเป็นศูนย์

เพื่อวัตถุประสงค์ในการวิเคราะห์ Esri จะคำนวณเส้นทางระบายไว้ล่วงหน้าโดยใช้รูปแบบอุทกวิทยามาตรฐาน หากไม่มีเส้นทางระบายน้ำอยู่ภายในระยะทางที่ค้นหา เครื่องมือจะใช้ตำแหน่งที่ที่มีการไหลสะสมสูงสุดภายในระยะทางค้นหา

ชื่อชั้นข้อมูลผลลัพธ์

ชื่อนี้เป็นชื่อของชั้นข้อมูลกลุ่มที่จะถูกสร้างอยู่ใน เนื้อหาของฉัน และจะถูกเพิ่มเข้าไปบนแผนที่ ชั้นข้อมูลกลุ่มจะประกอบด้วยชั้นข้อมูลของพื้นที่รับน้ำและจุดที่ถูกปรับ หากชั้นข้อมูลนั้นมีอยู่แล้ว คุณจะถูกถามเพื่อให้ยืนยันว่าคุณต้องการเขียนทับหรือไม่

เมื่อใช้กล่องคำสั่งแบบดรอปดาวน์ บันทึกผลลัพธ์ลงใน คุณสามารถกำหนดชื่อของโฟลเดอร์ใน เนื้อหาของฉัน ซึ่งเป็นที่จัดเก็บผลลัพธ์
