สรุปภายใน

เครื่องมือนี้ค้นหาฟีเจอร์ (และส่วนของฟีเจอร์) ที่อยู่ภายในขอบเขตของพื้นที่ในชั้นข้อมูลวิเคราะห์ For example:

  • Given a layer of watershed boundaries and a layer of land-use boundaries by land-use type, calculate total acreage of land-use type for each watershed.
  • มีชั้นข้อมูลที่ดินของเขตการปกครองหนึ่งและชั้นข้อมูลขอบเขตเมือง สรุปพื้นที่ของที่ดินว่างทั้งหมดที่อยู่ในแต่ละเมือง

หาก ใช้ขอบแขตแผนที่ปัจจุบันถูกเลือก จะมีเพียงฟีเจอร์ที่อยู่ภายในชั้นข้อมูลที่ต้องการสรุปที่มองเห็นภายในขอบเขตแผนที่ปัจจุบันเท่านั้นที่ถูกนำมาวิเคราะห์ หากไม่ถูกเลือก ทุกๆ ฟีเจอร์ในทั้งสองชั้นข้อมูลจะถูกวิเคราะห์ แม้ว่าฟีเจอร์นั้นจะอยู่นอกขอบเขตแผนที่ปัจจุบัน

เลือกชั้นข้อมูลผลรวม

ฟีเจอร์ของชั้นข้อมูลนี้ที่ตกอยู่ภายในขอบเขตของฟีเจอร์ในชั้นข้อมูลวิเคราะห์จะถูกนำมาสรุป

เพิ่มการคำนวณทางสถิติจาก

คุณสามารถคำนวณทางสถิติได้จากข้อมูลคุณลักษณะของข้อมูลจุดที่ตกอยู่ภายในขอบเขตของฟีเจอร์

หากเลือก จำนวนจุดที่กล่องตัวเลือก ข้อมูลจะถูกนำมาคำนวณหาจำนวนจุดทั้งหมดภายในแต่ละขอบเขต นอกจากนั้น คุณยังสามารถคำนวณทางสถิติจากข้อมูลคุณลักษณะของข้อมูลจุดที่อยู่ภายในแต่ละขอบเขตได้ด้วย

  • รวม—ค่ารวมของฟีลด์ที่เลือก
  • Average—The average or mean for the specified field.
  • Minimum—The smallest value in the specified field.
  • Maximum—The largest value in the specified field.
  • Std. Deviation—The standard deviation of the values in the specified field.

เพิ่มการคำนวณทางสถิติจาก

คุณสามารถคำนวณทางสถิติได้จากข้อมูลคุณลักษณะของฟีเจอร์เส้นที่อยู่ใกล้เคียง

กล่อง ความยาวของเส้นหากถูกเลือก จะคำนวณหาความยาวของฟีเจอร์เส้นทั้งหมดที่อยู่ภายในแต่ละขอบเขต นอกจากนั้น คุณยังสามารถคำนวณทางสถิติจากข้อมูลคุณลักษณะของข้อมูลเส้นที่อยู่ภายในแต่ละขอบเขตได้ด้วย

  • รวม—ค่ารวมของฟีลด์ที่เลือก
  • Average—The average or mean for the specified field.
  • Minimum—The smallest value in the specified field.
  • Maximum—The largest value in the specified field.
  • Std. Deviation—The standard deviation of the values in the specified field.

เพิ่มการคำนวณทางสถิติจาก

คุณสามารถคำนวณค่าทางสถิติได้จากข้อมูลคุณลักษณะของฟีเจอร์พื้นที่ที่อยู่ใกล้เคียง

กล่อง สรุปขนาดพื้นที่ที่ใช้หากถูกเลือก จะคำนวณพื้นที่ทั้งหมดของฟีเจอร์ที่อยู่ภายในขอบเขตแต่ละขอบเขตของชั้นข้อมูลวิเคราะห์ นอกจากนั้น คุณยังสามารถคำนวณทางสถิติจากข้อมูลเชิงบรรยายของฟีเจอร์ที่อยู่ภายในแต่ละขอบเขตได้ด้วย

  • รวม—ค่ารวมของฟีลด์ที่เลือก
  • Average—The average or mean for the specified field.
  • Minimum—The smallest value in the specified field.
  • Maximum—The largest value in the specified field.
  • Std. Deviation—The standard deviation of the values in the specified field.

เลือกฟิลด์ที่ต้องการให้รวมกลุ่มกัน

นี่คือข้อมูลคุณลักษณะของ การสรุปฟีเจอร์ที่คุณสามารถใช้คำนวณทางสถิติจากค่าในข้อมูลคุณลักษณะ ตัวอย่างเช่น ชั้นข้อมูลวิเคราะห์มีข้อมูลขอบเขตเมืองและฟีเจอร์ที่ต้องการ สรุปเป็นที่ดิน One of the attributes of the parcels is Status which contains two values: ว่าง และ ไม่ว่าง To calculate the total area of vacant and occupied parcels within the boundaries of cities, use Status as the group by attribute. ค่าสถิติของแต่ละกลุ่ม เช่นเดียวกับจำนวนฟีเจอร์ที่อยู่ภายในขอบเขตแต่ละพื้นที่ จะถูกนำมาแสดงบน pop-up ของชั้นข้อมูลผลลัพธ์

เพิ่มกลุ่มน้อย กลุ่มมาก

This checkbox is enabled when you choose a field to group by. If you want to find out which attribute values within each group are the minority (least dominant) or the majority (most dominant) within each boundary of the analysis layer, select Add minority, majority. เมื่อคุณเลือก เพิ่มกลุ่มน้อย กลุ่มมาก ฟิลด์อีกสองฟิลด์จะถูกเพิ่มลงในชั้นข้อมูลผลลัพธ์ของคุณ หากคุณเลือก เพิ่มเปอร์เซ็นต์ ด้วย ฟิลด์อีกสองฟิลด์จะถูกเพิ่มลงในชั้นข้อมูลผลลัพธ์ที่ประกอบด้วยเปอร์เซ็นต์ของค่าข้อมูลคุณลักษณะส่วนน้อยและส่วนมากภายในแต่ละกลุ่ม

  • If you are summarizing points, minority and majority values are calculated based on the count of points within each bounding area in the analysis layer.
  • If you are summarizing lines, minority and majority values are calculated based on the length of lines within each bounding area in the analysis layer.
  • If you are summarizing areas, minority and majority values are calculated based on the areas within each bounding area in the analysis layer.

เพิ่มเปอร์เซนต์

คุณจะใช้กล่องกาเครื่องหมายนี้ได้หากคุณเลือกฟิลด์ที่จะจัดกลุ่ม เลือก เพิ่มเปอร์เซนต์ หากคุณต้องการค้นหาเปอร์เซ็นต์ของค่าตัวแปรแต่ละค่าภายในแต่ละกลุ่ม ฟิลด์ใหม่จะถูกเพิ่มลงในตารางผลลัพธ์ที่ประกอบด้วยเปอร์เซ็นต์ของค่าตัวแปรแต่ละค่าภายในแต่ละกลุ่ม หาก เพิ่มส่วนน้อย ส่วนใหญ่ ก็จะถูกเลือกด้วย ฟิลด์อีกสองฟิลด์จะถูกเพิ่มลงในชั้นข้อมูลผลลัพธ์ที่ประกอบด้วยเปอร์เซ็นต์ของค่าตัวแปรส่วนน้อยและส่วนใหญ่ภายในแต่ละกลุ่ม

ชื่อชั้นข้อมูลผลลัพธ์

ชื่อนี้เป็นชื่อของชั้นข้อมูลที่จะถูกสร้างอยู่ใน เนื้อหาของฉัน และจะถูกเพิ่มเข้าไปบนแผนที่ ชื่อตั้งต้นเชื่อมโยงกับชื่อชั้นข้อมูลที่ใช้วิเคราะห์ หากชั้นข้อมูลนั้นมีอยู่แล้ว คุณจะถูกถามเพื่อให้ยืนยันว่าคุณต้องการเขียนทับหรือไม่

เมื่อใช้กล่องคำสั่งแบบดรอปดาวน์ บันทึกผลลัพธ์ใน คุณสามารถกำหนดชื่อของโฟลเดอร์ใน เนื้อหาของฉัน ซึ่งเป็นที่จัดเก็บผลลัพธ์
